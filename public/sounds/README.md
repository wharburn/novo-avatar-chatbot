# Camera Click Sound

## Required File

You need to add a camera click sound file to this directory:

**File:** `camera-click.mp3`

## Where to Get a Camera Click Sound:

### Option 1: Free Sound Libraries
- **Freesound.org**: https://freesound.org/search/?q=camera+shutter
- **Zapsplat**: https://www.zapsplat.com/sound-effect-category/cameras/
- **Mixkit**: https://mixkit.co/free-sound-effects/camera/

### Option 2: Use a Simple Beep
You can use any short sound effect (0.5-1 second) as a placeholder.

### Option 3: Generate Your Own
Use audio editing software like Audacity to create a simple click sound.

## File Requirements:

- **Format:** MP3
- **Duration:** 0.5 - 1 second
- **Size:** < 50KB recommended
- **Name:** `camera-click.mp3` (exact name)

## Installation:

1. Download a camera shutter sound
2. Save it as `camera-click.mp3`
3. Place it in this directory: `public/sounds/camera-click.mp3`
4. The app will automatically use it when taking pictures

## Testing:

After adding the file, test it by:
1. Running the app
2. Saying "Take a picture"
3. You should hear the click sound when the photo is captured
